## 專案概覽

一個專為室內設計工作室量身打造的全方位專案管理平台，旨在簡化專案工作流程、客戶溝通與資源管理。

## 功能特色

### 專案管理

- 專案時間軸與里程碑
- 任務指派與追蹤
- 資源分配
- 預算管理
- 進度報告

### 檔案管理

- 設計檔案版本控制
- 客戶文件儲存
- 共享檔案存取
- 雲端儲存整合
- 常用檔案格式預覽

### 客戶端平台

- 專案狀態檢視
- 設計簽核流程
- 直接溝通管道
- 發票與付款追蹤

### 團隊協作

- 即時更新
- 留言與回饋系統
- 基於角色的權限管理
- 活動通知

## 技術堆疊

### 前端

- **框架**：Vue 2
- **UI 元件**：Element UI
- **狀態管理**：Vuex
- **HTTP 客戶端**：Axios

### 後端

- **框架**：ASP.NET Core 3.1
- **API**：RESTful Web API
- **驗證**：JWT (JSON Web Tokens)
- **資料庫 ORM**：Entity Framework Core

### 資料庫

- **關聯式資料庫**：Microsoft SQL Server
- **架構**：正規化關聯設計
- **索引**：查詢效能最佳化

### 基礎設施

- **檔案儲存**：Azure Blob Storage
- **快取**：Redis
- **部署**：IIS / Azure App Service

## 關鍵功能實作

### 任務管理

- 拖放式看板 (Kanban)
- 甘特圖時間軸檢視
- 任務相依性
- 自動化狀態更新

### 檔案版本控制

- 自動版本控制
- 版本比較
- 版本回溯功能
- 變更歷史追蹤

### 即時通知

- 使用 SignalR 進行推播通知
- 重要事件的 Email 提醒
- 應用程式內通知中心
- 可自訂的通知偏好

## 商業效益

- 降低 35% 的專案管理負擔
- 提升客戶滿意度
- 簡化溝通流程
- 增進團隊協作效率
- 集中化管理專案文件
